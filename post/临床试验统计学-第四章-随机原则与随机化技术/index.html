临床试验统计学 第四章 随机原则与随机化技术

随机原则是临床试验必须遵循的三个基本原则之一。

遵循随机原则可以尽量避免受试者的选择偏倚；同时，只有合乎随机原则的数据才能正确应用数理统计的各种分析方法。 随机对照是临床试验的一种“金标准”方法，随机化技术和盲法技术的结合是保证临床试验非 处理因素在组间分布均衡的重要措施。 广义地讲，科学研究中随机化包含了三个方面的内容：

①抽样随机（random sampling):每一个符合条件的人参加试验的机会相同，即总体中的每个个体有相同的机会被抽到试验 样本中来；

②分组随机（random allocation):每个实验对象分配到某处理组的机会相同；

③试验顺序随机：每个受试者先后接受处理的机会相同。在临床试验中，很难或不可能做到随机 抽样。因此，临床试验中关注的是分组的随机。

本章所介绍的内容为随机化分组。

第一节 随机化及其意义

  1. 1662年 随机化思想的提出 成组随机
  2. 1898年 交替分组法 基于受试者个体的分组 （事先知道下一个受试者的分组，无法避免选择性偏倚）
  3. 1948年 真正现代化意义的临床随机化分组

随机化分组的意义

随机化分组：分配到每个组的机会均等、不收主观意愿影响、避免系统差异

意义：

1. 各处理组的各种非处理因素分布趋于相似，组间基线具有可比性
2. 应用假设检验进行比较分析的前提

随机化分组方法的分类

按照随机化分组的概率在试验中是否保持一致，随机化可以分为固定随机化方法和动态随机化

固定随机法：按照事先确定的概率讲受试者分配到不同的组别，整个研究过程中保持分配概率不变。
简单随机化、区组随机化分层随机化、分层区组随机化

动态随机法：适用于样本量较小，又必须考虑某些重要的分层因素，如抗肿瘤药物试验
试验过程中受试者随机化分组的概率根据入组的受试者情况而变化。
偏币法、瓮法、最小化法

本章介绍简单随机化、区组随机化、分层随机化、区组分层随机化、最小随机化
其他适应性随机化方法见二十章

第二节 简单随机化

也称为完全随机化。除了对受试者的数量及组间分配比例有所要求外，对随机化序列不附加任何限制的随机化过程。

简单随机化法包括：抽签法、掷硬币法、排序法、随机数字表法。

目前医学研究中的随机化采用计算机程序产生伪随机数进行

/*240例患者，按照1：1的比例分配，采用完全随机方法分成两组。*/

/*输出数据集rand,r:1-240的随机数序列*/
proc plan seed = 210002;
    factors r = 240;
    output out = rand;
run;

/*根据随机数进行1：1比例分配*/
data rand;
    set rand;
    number = _n_;
    if r <= 120 then group = 'A';
    else group = 'B';
run;

/*输出随机化的分配结果*/
proc print noobs;
    var number group;
run;

UNIFORM Function

data demo;
	do i = 1 to 240;
		r = uniform(2100001);
		output;
	end;
run;

proc rank data=demo out=rank ;
	ranks r_rank;
	var r;
run;

data rank1;
	set rank;
	if _n_ >=120 then group = 'A';
	else group = 'B';
run;

proc print data = rank1;
	var r_rank group;
run;

第三节 区组随机化

也称为：均衡随机化、限制性随机化
与简单随机化相比，可以确保在整个实验过程中进入每一组的受试者数基本相等，避免分配进度存在时间上的快慢差异，将偏倚减少到最小。
区组随机化是常用的随机化方法。

  1. 确定区组长度（区组中对象的数目，组数的2-3倍）
  2. 在区组内按照事先确定的分配比例随机分配

优点：随机化的过程中，时刻保证各组人数的平衡。某一阶段入组受试者特征发生变化，区组随机化可以获得更好的可比性分组。
隐患：医生可以猜到每一区组最后一位受试者属于哪一组（4人一区组，前三位ABB，最后必定为B）。区组越小越容易预测，应尽量避免只有两位受试者的区组，采用可变区组。 此处有疑问：盲态情况下，如何得知前面受试者分组的？

/*240例患者，按照2：1比例分配，根据区组随机化分组*/
/*生成随机数序列，区组长度取6，分为40组*/
proc plan seed = 200001;
	factors block = 40 length = 6;
	output out = rand;
run;

/*每个区组前4人设为A组，后2人设为B组*/
data rand;
	set rand;
	subjid = _n_;
	if length <= 4 then group = 'A';
	else group = 'B';
	keep subjid group;
run;
/*输出随机化结果*/
proc print data = rand noobs;
run;

第四节 分层区组随机化

优点：分层可以保证组内各层之间的均衡性，区组可以保证组间的可比性。

假设试验方案中，受试者年龄与性别的不同，对疾病的预后影响很大，需要按照不同的年龄与性别进行组合分层。

将产生4个亚组：①年龄<18岁的男性、②年龄≥18岁的男性、③年龄<18岁的女性、④年龄≥18岁的女性。

多中心临床试验中，中心是常见的分层因素，先按照中心分层，再根据其他因素分层，保证每个中心都能够随机化分组。

注意点：

  1. 随机化方法的科学性。

    选取正确的分层因素，入院日期（奇偶数分组）不应作为分层因素。

  2. 保证随机化的准确性。

    确保随机数序列不被破坏，严格按照试验用药品编号的先后顺序入组。

  3. 保证区组长度是试验组数的整数倍以上。
